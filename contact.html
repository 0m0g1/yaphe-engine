Support

If you have a problem learning Yaphe or need help, please first use the various community channels. Yahpe developers are busy working in making Yahpe better, so they can't always answer questions. They will, however, be glad to assist you if you can't get an answer from conventional means.

Commertial support

While we don't have a formal structure for offering commercial support, many Yahpe developers can already help you, work per hour, or relocate to your company. For such inquiries, contact support@Yahpeengine.org. Let us know about your needs and we'll try to find the right person for you.

Security Vulnerabilities

If you've found a security vulnerability in Yahpe, please do not create an issue on the GitHub issue tracker as issues are visible publicly.

Instead, send an email to security@Yahpeengine.org.

General

For other inquiries such as business, education, non-profit, promotion, etc. Contact contact@Yahpeengine.org.
